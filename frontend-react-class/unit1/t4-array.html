1. concat() Merges two or more arrays and returns a new array. const arr1 = [1, 2]; const arr2 = [3, 4]; const merged = arr1.concat(arr2); console.log(merged); // [1, 2, 3, 4] ✅ Does not change the original arrays. ============================================================= 2. every() Checks if all elements pass a test (returns true or false). const nums = [2, 4, 6]; const allEven = nums.every(num => num % 2 === 0); console.log(allEven); // true ✅ Stops checking as soon as a value fails. ==================================================================== 3. filter() Creates a new array with elements that pass a test. const ages = [15, 20, 25, 30]; const adults = ages.filter(age => age >= 18); console.log(adults); // [20, 25, 30] ✅ Does not modify the original array. ======================================================================= 4. find() Returns the first element that passes a test. const numbers = [5, 12, 8, 130, 44]; const found = numbers.find(num => num > 10); console.log(found); // 12 ✅ Stops after the first match. ========================================================================= 5. forEach() Runs a function for each array element. const fruits = ["apple", "banana", "mango"]; fruits.forEach(fruit => console.log(fruit.toUpperCase())); // Output: APPLE, BANANA, MANGO ✅ Does not return a new array. ========================================================================= 6. Array.from() Creates an array from an iterable (like a string). const str = "hello"; const letters = Array.from(str); console.log(letters); // ['h', 'e', 'l', 'l', 'o'] ✅ Can also take a mapping function: const numsArray = Array.from([1, 2, 3], x => x * 2); console.log(numsArray); // [2, 4, 6] ============================================================================= 7. indexOf() Returns the first index of a value (or -1 if not found). const colors = ["red", "blue", "green", "blue"]; console.log(colors.indexOf("blue")); // 1 console.log(colors.indexOf("yellow")); // -1 ============================================================================ 8. map() Creates a new array by applying a function to each element. const nums2 = [1, 2, 3]; const squared = nums2.map(num => num ** 2); console.log(squared); // [1, 4, 9] ============================================================================ 9. reduce() Reduces array to a single value by applying a function. const values = [1, 2, 3, 4]; const sum = values.reduce((total, num) => total + num, 0); console.log(sum); // 10 ✅ 0 is the initial value here. ===================================================================== 10. reverse() Reverses the order of elements in place. const arr3 = [1, 2, 3]; arr3.reverse(); console.log(arr3); // [3, 2, 1] ✅ Modifies the original array.
