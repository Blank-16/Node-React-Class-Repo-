What Are JavaScript Modules? A module is a file that contains reusable code. Each module can export variables, functions, or classes. Other files can import them. Helps in code organization, reusability, and maintainability. 1. Default export: One main export per file (imported without {}). 2. Importing a Module ðŸ“„ main.js // Import named exports import { add, multiply } from './math.js'; // Import default export import subtract from './math.js'; console.log(add(5, 3)); // 8 console.log(multiply(5, 3)); // 15 console.log(subtract(5, 3)); // 2 Explanation: { add, multiply } matches the names from math.js. Default export (subtract) can be given any name when importing. 3.
